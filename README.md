# TrotskysWorld
a shitty version of a shittier software to test the basics of logic

## RULES
* tags and variables must be placed between inverted commas e.g. Square('a')
* everything that follows "exist" or "for each" goes between brackets e.g. $x(Square('x'))

## ATOMIC SENTENCES
at the moment these are the aviable atomic sentences
* Square('tag')
* Triangle('tag')
* Circle('tag')
* SameShape('tagA', 'tagB')
* Small('tag')
* Medium('tag')
* Large('tag')
* Smaller('tagA', 'tagB')
* Larger('tagA', 'tagB')
* SameSize('tagA', 'tagB')
* SameRow('tagA', 'tagB')
* SameCol('tagA', 'tagB')
* LeftOf('tagA', 'tagB')
* RightOf('tagA', 'tagB')
* Over('tagA', 'tagB')
* Below('tagA', 'tagB')

## SHORTCUT
* ! = not
* $ = for each
* % = exist
* & = and
* | = or
